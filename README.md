# HR Attrition Analysis

## Problem Statement
Analyzed IBM HR dataset to identify key drivers of employee attrition.

## Tools Used
Python (Pandas, SQLite, Matplotlib, Seaborn), SQL, PowerBI

## Key Findings
- Overall attrition rate: 16.12%
- Sales department had the highest attrition (20.63%)
- Employees earning <$4k/month were 3x more likely to leave
- Employees under 30 showed the highest attrition rate
